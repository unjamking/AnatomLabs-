# Assets Directory

This directory contains app assets for the mobile application.

## Required Assets

To complete the app setup, add the following image files:

1. **icon.png** (1024x1024) - App icon
2. **splash-icon.png** (1242x2436) - Splash screen image  
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon
4. **favicon.png** (48x48) - Web favicon

## Temporary Placeholder

For development purposes, you can use a solid color image or create simple placeholder icons.

To generate placeholder icons quickly:
- Use online tools like https://icon.kitchen/
- Or create simple colored squares in any image editor
- Recommended colors: #e74c3c (red) or #0a0a0a (dark)

## Production

For production builds, replace these with professionally designed assets that match your brand identity.
